Este diário de bordo tem como objetivo registrar o processo de desenvolvimento de um site com 10 páginas, utilizando as linguagens HTML e CSS, como parte da disciplina de front-end.
O trabalho foi realizado em grupo, dividido entre nove integrantes, onde cada participante ficou responsável pela criação e estilização de determinadas páginas. 
Além da construção individual, o grupo trabalhou em conjunto para manter um padrão visual, estrutural e de navegação entre as páginas.

Data: 30/08/25

Atividades realizadas:

Reunião inicial do grupo para definição do tema do site e divisão das páginas.
Escolhemos fazer o site da barbearia de um amigo nosso.
Cada integrante ficará responsável por criar páginas específicas

Data: 04/09/25

Atividades realizadas:Em sala, dividimos o projeto em partes iguais para que cada integrante faça pelo menos uma pagina do html.

Dificuldades encontradas:

A principio foi dificil encaixar o projeto na rotina de trabalho e estudos dos integrantes do grupo pensando em não sobrecarregar o pessoal.

Próximos passos:

iniciar o desenvolvimento.

Data: 16/10/25

Atividades realizadas:

Todas as paginas html finalizadas, integradas e linkadas umas as outras cumprindo os requisitos pedidos pelo professor.

Dificuldades encontradas:

Próximos passos:

Conclusão:

O desenvolvimento deste projeto permitiu colocar em prática os conhecimentos de html e css, além de reforçar habilidades de trabalho em equipe, organização e padrão de código.
Durante o processo, aprendrendemos a resolver problemas de alinhamento, estrutura e estilo, entendendo melhor como cada propriedade CSS afeta o layout. Também percebemos a importância da comunicação e da padronização entre os membros do grupo para alcançar um bom resultado visual.
